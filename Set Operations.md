# Set Operations:
Using notation from [[DIscrete Structure Notations]]
- ## Complement:
	- $\overline{A}$
	- The set of all elements NOT in A
	- {$X : X \notin A$}
- ### Set Difference:
	- A\\B
		- {$X : X \in A, X \notin B$}
		- All elements in A, not in B
- ## Symmetric Difference:
	- $A \Delta B$
	- {$X : X \notin A\cap B$}
	- Elements in A XOR B
	- $(A-B)\cup(B-A)$
	- $A\cup B - A\cap B$
- ## Cartesian Product:
	- #### Set of all lists pairs formed with one element of A and one element of B...
		- Order of elements in lists dictated by order of set appearance in the product expression
		- Similar to linear combinations
		- AxB = {$(a,b) : a\in A, b\in B$} 
		- BxA = {$(b,a) : a\in A, b\in B$}
